# FarmWhisper Tools

Standalone tools and utility firmware for FarmWhisper development.

## Tools

### heltec-v4-packet-tool

Simple Heltec WiFi LoRa 32 V4 packet monitor.

It listens for FarmWhisper LoRa P2P traffic and prints every valid received packet to:

- USB serial
- Built-in OLED display

This is not a full FarmWhisper node. It is a field/debug tool.
